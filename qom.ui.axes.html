class qom.ui.axes.BaseAxis.BaseAxis(params)¶

Bases: object

Class to interface axes.

Initializes val, dim, bound, ticks and tick_labels properties. Inherited objects need to set the other properties individually.

Parameters

params (dict) – Data for the axis.

property bound¶

Option to check user-defined bounds: * ‘none’ : Not bounded. * ‘lower’ : Lower bound. * ‘upper’ : Upper bound. * ‘both’ : Both lower and upper bound.
