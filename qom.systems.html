get_mean_optical_amplitude(lambda_l: Union[float, numpy.float64], mu: Union[float, numpy.float64], gamma_o: Union[float, numpy.float64], P_l: Union[float, numpy.float64], Delta: Union[float, numpy.float64], C: Union[float, numpy.float64] = 0, mode='basic')¶

Method to obtain the mean optical amplitude.

The optical steady state is assumed to be of the form [1],

\[\alpha_{s} = \frac{\eta_{l}}{\frac{\gamma_{o}}{2} - \iota \Delta}\]

where \(\Delta = \Delta_{l} + C |\alpha_{s}|^{2}\), \(\eta_{l} = \sqrt{\mu \gamma_{o} P_{l} / (\hbar \omega_{l})}\), with \(\omega_{l} = 2 \pi c / \lambda_{l}\).

Parameters

  • lambda_l (float or numpy.float) – Wavelength of the laser.

  • mu (float or numpy.float) – Laser-cavity coupling parameter.

  • gamma_o (float or numpy.float) – Optical decay rate.

  • P_l (float or numpy.float) – Power of the laser.

  • Delta (float or numpy.float) – Effective detuning if mode is ‘basic’, else detuning of the laser.

  • G (float or numpy.float) – Frequency pull parameter.

  • C (float or numpy.float, optional) – Coefficient of \(|\alpha_{s}|^{2}\).

  • mode (str, optional) –

    Mode of calculation of intracavity photon number:

    • ’basic’ : Assuming constant effective detuning.

    • ’cubic’ : Cubic variation with laser detuning.

Returns

alpha_s – Mean optical amplitude.

Return type

float or numpy.float or list

get_mean_optical_occupancy(lambda_l: Union[float, numpy.float64], mu: Union[float, numpy.float64], gamma_o: Union[float, numpy.float64], P_l: Union[float, numpy.float64], Delta: Union[float, numpy.float64], C: Union[float, numpy.float64] = 0, mode='basic')¶

Method to obtain the mean optical occupancy.

The mean optical occupancy can be written as [1],

\[N_{o} = |\alpha_{s}|^{2} = \frac{\eta_{l}^{2}}{\frac{\gamma_{o}^{2}}{4} + \Delta^{2}}\]

where \(\Delta = \Delta_{l} + C N_{o}\), \(\eta_{l} = \sqrt{\mu \gamma_{o} P_{l} / (\hbar \omega_{l})}\), with \(\omega_{l} = 2 \pi c / \lambda_{l}\).

Parameters

  • lambda_l (float or numpy.float) – Wavelength of the laser.

  • mu (float or numpy.float) – Laser-cavity coupling parameter.

  • gamma_o (float or numpy.float) – Optical decay rate.

  • P_l (float or numpy.float) – Power of the laser.

  • Delta (float or numpy.float) – Effective detuning if mode is ‘basic’, else detuning of the laser.

  • C (float or numpy.float, optional) – Coefficient of \(N_{o}\) in the expression of \(\Delta\).

  • mode (str, optional) –

    Mode of calculation of intracavity photon number:

    • ’basic’ : Assuming constant effective detuning.

    • ’cubic’ : Cubic variation with laser detuning.

Returns

N_o – Mean optical amplitude.

Return type

float or numpy.float or list

get_measure_pearson(solver_params: dict, ode_func, get_ivc, ode_func_corrs=None, plot: bool = False, plotter_params: dict = None)¶

Method to obtain the Pearson synchronization measure.

The implementation measure reads as [2],

\[{S_{Pearson}}_{ij} = \frac{\Sigma_{t} \langle \delta O_{i} (t) \delta O_{j} (t) \rangle}{\sqrt{\Sigma_{t} \langle \delta O_{i}^{2} (t) \rangle} \sqrt{\Sigma_{t} \langle \delta O_{j}^{2} (t) \rangle}}\]

where \(\delta O_{i}\) and \(\delta O_{j}\) are the corresponding quadratures of fluctuations.

Parameters

  • solver_params (dict) – Parameters for the solver.

  • ode_func (function) – Set of ODEs returning rate equations of the classical mode amplitudes and quantum correlations. If ode_func_corrs parameter is given, this function is treated as the function for the modes only.

  • get_ivc (function) – Function returning the initial values and constants.

  • ode_func_corrs (function, optional) – Set of ODEs returning rate equations of the quantum correlations.

  • plot (bool, optional) – Option to plot the measures. Requires plotter_params parameter if True.

  • plotter_params (dict, optional) – Parameters for the plotter.

Returns

S_Pearson – Pearson synchronization measure.

Return type

float
