  • cache (bool, optional) – Option to cache the dynamics.

  • cache_dir (str, optional) – Directory where the results are cached.

  • cache_file (str, optional) – File where the results are cached.

qom.solvers.RHCSolver module¶

Class to handle Routh-Hurwitz criterion solver.

class qom.solvers.RHCSolver.RHCSolver(A: Union[list, numpy.matrix, numpy.ndarray] = None, coeffs: Union[list, numpy.matrix, numpy.ndarray] = None)¶

Bases: object

Class to handle Routh-Hurwitz criterion solver.

Initializes coeffs, n and seq properties. At least one of the parameters, preferably “coeffs”, should be non-None.

An eigenvalue equation in \(\lambda\) is obtained by equating \(\det(\lambda I_{n} - A_{n \times n})\) to zero.

References:

[1] E. X. DeJesus and C. Kaufman, Routh-Hurwitz Criterion in the Examination of Eigenvalues of a System of Nonlinear Ordinary Differential Equations, Phys. Rev. A 35 (12), 5288 (1987).

Parameters

  • A (list or numpy.matrix or numpy.ndarray, optional) – Drift matrix.

  • coeffs (list or numpy.ndarray, optional) – Coefficients of the characteristic equation.

code = 'rhc'¶

name = 'Routh Hurwitz Criterion Solver'¶
property seq¶

Sequence of terms \(T_{k} = \det(M_{k})\), where \(M_{k}\) is the square sub-matrix of \(M\) comprising of its first \(k\) rows and columns, with \(T_{0} = a_{0}\).

The matrix \(M\) is defined as [1],

\[\begin{split}M_{ij} = \begin{cases} a_{2i - j}, ~ \mathrm{if} ~ 0 \le 2 i - j \le n \\ 0, ~ \mathrm{otherwise} \end{cases}\end{split}\]

where the indices of \(M\) are 1-based.

Type

list or numpy.ndarray
