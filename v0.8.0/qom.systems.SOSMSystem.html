qom.systems.SOSMSystem module¶

Module to interface a single-optical single-mechanical system.

class qom.systems.SOSMSystem.SOSMSystem(params, cb_update=None)¶

Bases: qom.systems.BaseSystem.BaseSystem

Class to interface a single-optical single-mechanical system.

Parameters

  • params (dict) – Parameters for the system.

  • cb_update (callable, optional) – Callback function to update status and progress, formatted as cb_update(status, progress, reset), where status is a string, progress is an integer and reset is a boolean.

  • note: (.) – All the options defined in params supersede individual function arguments. Refer qom.systems.BaseSystem for a complete list of supported options.:
