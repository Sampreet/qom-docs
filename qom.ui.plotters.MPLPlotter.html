class qom.ui.plotters.MPLPlotter.MPLPlotter(axes: dict, params: dict)¶

Bases: BasePlotter

Class to handle matplotlib plots.

Parameters:

  • axes (dict) – Axes for the plot containing one or more keys for the axes (“X”, “Y” or “Z”), each either a list of values, or dictionary containing “min”, “max” and “dim” keys or a single “val” key with a non-empty list. Refer qom.ui.axes.BaseAxis for all currently supported keys.

  • params (dict) – Parameters of the plot. Refer qom.ui.plotters.BasePlotter for all currently supported keys.

  • mpl_spec (matplotlib.gridspec.GridSpec) – Grid for the plot.

  • note: (..) – All the options defined in params supersede individual function arguments. Refer qom.ui.plotters.BasePlotter for a complete list of supported options.:
