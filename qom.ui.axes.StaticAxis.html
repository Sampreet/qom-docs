qom.ui.axes.StaticAxis module¶

Module to handle a static axis.

class qom.ui.axes.StaticAxis.StaticAxis(axis, axis_params, plotter_params)¶

Bases: BaseAxis

Class to handle a static axis.

Parameters:

  • axis (str) – Name of the axis, “X”, “Y”, “Z” or “V”.

  • axis_params (dict or list) – Values for the axis supporting a list of values or a dictionary containing the range of values with keys “min”, “max”, “dim” and “scale” or the values themselves under key “val”.

  • plotter_params (dict) –

    Parameters for the plotter. Along with the keys defined in qom.ui.axes.BaseAxis, additionally supported keys are:

    key

    value

    ”name”

    (str) display name of the axis. Default is ‘StaticAxis’.

    ”unit”

    (str) unit of the plots.
